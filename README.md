# Digital Literacy Portfolio

## Research Question
How is organising information different from synthesising it, and why does synthesis matter more for academic work?

## Key Findings (Chapter 4: Organise & Synthesise)

1. **Organising is not synthesising.** Organising just sorts and files what you found (like folders on a desktop), so you know where things are. Synthesising combines ideas from different sources into new understanding — this is what's actually required for academic and professional work.

2. **A synthesis matrix reveals 3 patterns.** Putting themes as rows and sources as columns helps you spot consensus (sources agree), contradiction (sources disagree), and gaps (no source covers it) — contradictions and gaps are often the most interesting findings.

3. **Record sources immediately, not later.** Citation details, quotes, and your own paraphrases should be recorded as soon as you find them, using tools like Notion or Zotero. Recording the source always comes before recording the content.
